# Release Test Notes - 0.4.1

Date: 2026-06-02

## Scope

This release aligns the public ABI, Python binding, C++ binding, and
documentation API surface at version `0.4.1`.

The release adds ABI metadata discovery, completes C++ wrapper parity with
Python for RobStride helper methods, and introduces `bindings/api_surface.json`
as the canonical binding/API checklist.

## Key Changes

- Added C ABI metadata functions:
  - `motor_abi_version()`
  - `motor_abi_capabilities_json()`
- Added Python metadata helpers:
  - `motorbridge.abi_version()`
  - `motorbridge.abi_capabilities()`
- Added C++ metadata helpers:
  - `motorbridge::abi_version()`
  - `motorbridge::abi_capabilities_json()`
- Completed C++ RobStride wrapper parity for:
  - `robstride_ping_host_id`
  - `robstride_get_param_f32_host_id`
  - `robstride_get_fault_report`
  - `robstride_set_active_report`
- Added C++ RobStride ID range validation matching Python behavior.
- Added `bindings/api_surface.json` and API surface regression coverage.

## Required Local Checks

```bash
cargo fmt --all -- --check
cargo check -p motor_abi
cargo build -p motor_abi
PYTHONPATH=bindings/python/src python3 -m compileall bindings/python/src/motorbridge
```

Python API surface and metadata smoke:

```bash
MOTORBRIDGE_LIB=target/debug/libmotor_abi.so \
PYTHONPATH=bindings/python/src \
python3 - <<'PY'
import motorbridge

assert motorbridge.get_version() == "0.4.1"
assert motorbridge.abi_version() == "0.4.1"
caps = motorbridge.abi_capabilities()
assert caps["abi"]["version"] == "0.4.1"
assert "dm-serial" in caps["transports"]
assert "robstride" in caps["vendors"]
print(motorbridge.__version__)
print(caps["abi"]["name"], caps["abi"]["version"])
PY
```

Windows equivalent:

```powershell
$env:MOTORBRIDGE_LIB=(Resolve-Path 'target/debug/motor_abi.dll')
$env:PYTHONPATH=(Resolve-Path 'bindings/python/src')
python - <<'PY'
import motorbridge
assert motorbridge.get_version() == "0.4.1"
assert motorbridge.abi_version() == "0.4.1"
caps = motorbridge.abi_capabilities()
assert caps["abi"]["version"] == "0.4.1"
print("abi metadata ok")
PY
```

## C++ Binding Notes

C++ users can now query ABI metadata and call the same RobStride helper methods
already available in Python. The wrapper remains header-only and compatible with
the existing `motor_abi` C interface plus the new read-only metadata functions.
