# media

Repository media used by the top-level README.

Adapter photos are already referenced from the top-level README. Motor images
are not referenced yet; add them later when real device photos are available.

Suggested layout:

```text
media/
  adapters/
    canable.png
    pcan.png
    dm-usb2can.jpg
    dm-usb2canfd.jpg
    dm-usb2canfd-dual.jpg
    DM-LinkX-4C.jpg
```

Image guidance:

- Use project-owned photos or images with clear redistribution rights.
- Keep README images small; around 300 KB to 800 KB each is a good target.
- Avoid committing large raw photos. Use Git LFS only if large binary media
  becomes necessary.
